Szoftverleltár beadandó

A projekt jelenleg már működő JavaScript, React, SPA, Fetch API, Axios és OOJS blokkokat tartalmaz. A közös adatkör továbbra is a szoftverleltár, technikai oldalról már csak a végső ellenőrzés, a dokumentáció és a screenshotok összeállítása maradt hátra.

SPA rész Axios rész

JavaScript CRUD

Lokális táblázat, űrlap, szűrés és alapadatkezelés.

React CRUD

Komponens alapú újraépítés ugyanarra az adatkészletre.

SPA

Két miniapp közös állapottal, hash-alapú navigációval egyetlen React SPA-n belül.

Fetch API

PHP API-ra kapcsolt szerveroldali CRUD.

Axios

React + Axios kliens ugyanarra a backendre.

OOJS

Objektumorientált JavaScript nézet vagy grafikus demó.
